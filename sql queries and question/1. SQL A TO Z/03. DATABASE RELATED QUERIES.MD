# Database related Queries

- `CREATE THE DATABASE`

```sql
CREATE DATABASE db_name;

CREATE DATABSE IF NOT EXISTS db_nmae;

CREATE DATABSE IF NOT EXISTS COLLEGE;
```

- `DROP THE DATABASE`

```sql
DROP DATABASE db_name;

DROP DATABASE IF EXISTS db_name;
```

- `SHOW THE DATABASEN & TABLE`

```sql
SHOW DATABASES;

SHOW TABLES;
```
